    ToGpxRoute(IFeature)

    Turns an GeoAPI.Geometries.ILineString NetTopologySuite.Features.IFeature into the GpxRoute instance that's equivalent to it.

    Declaration
    public static GpxRoute ToGpxRoute(IFeature feature)
    Parameters
    Type Name Description
    NetTopologySuite.Features.IFeature feature

    The NetTopologySuite.Features.IFeature to convert.
    Returns
    Type Description
    GpxRoute

    A more-or-less equivalent GpxRoute to feature.
    Remarks

    Values from NetTopologySuite.Features.IFeature.Attributes with keys whose names match the names of properties of GpxRoute will be used to populate those corresponding property values.

    When the "Waypoints" attribute is populated in NetTopologySuite.Features.IFeature.Attributes, its values will be used for Waypoints, and the NetTopologySuite.Features.IFeature.Geometry will be ignored except to validate that it is, in fact, GeoAPI.Geometries.ILineString. Otherwise, waypoints will be built with only values for:

    • Longitude,
    • Latitude, and
    • ElevationInMeters (where GeoAPI.Geometries.IPoint.Z is set).

    Exceptions
    Type Condition
    ArgumentNullException

    Thrown when feature is null.
    ArgumentOutOfRangeException

    Thrown by GpxLongitude(Double) or GpxLatitude(Double).
    ArgumentException

    Thrown when feature's NetTopologySuite.Features.IFeature.Geometry is not an instance of GeoAPI.Geometries.ILineString.

    Values from NetTopologySuite.Features.IFeature.Attributes with keys whose names match the names of properties of GpxTrack will be used to populate those corresponding property values.

    When the "Segments" attribute is populated in NetTopologySuite.Features.IFeature.Attributes, its values will be used for Segments, and the NetTopologySuite.Features.IFeature.Geometry will be ignored except to validate that it is, in fact, GeoAPI.Geometries.IMultiLineString.     • ElevationInMeters (if GeoAPI.Geometries.ICoordinateSequence.Ordinates has the GeoAPI.Geometries.Ordinates.Z flag).
